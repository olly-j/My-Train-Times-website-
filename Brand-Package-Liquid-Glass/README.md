# TubeBoard Liquid Glass Brand Package

Created: 2026-06-12

This package upgrades TubeBoard from a starter black/amber asset set into a premium iOS Liquid Glass-inspired brand system for an independent London Tube live arrivals app.

## Contents

- `01-Asset-Audit/asset-audit.md` - Existing asset review and priority improvements.
- `02-Brand-Guidelines/brand-guidelines.md` - Brand essence, colour, typography, layout, glass, and motif rules.
- `03-App-Icon/` - App icon system documentation, v2 SVG source, generated concept reference, and PNG exports.
- `04-App-Store/app-store-creative.md` - Listing copy, screenshot system, ASO, and preview video direction.
- `05-Website/` - Website brief, copy, and visual direction.
- `06-Social-Ads/social-ad-kit.md` - Launch campaign, social copy, ad headlines, and motion scripts.
- `07-Press-Kit/press-kit.md` - Press kit, factsheet, release, and outreach copy.
- `08-Production-System/` - Layer system, naming, export specs, and production checklist.
- `09-Legal-Trust/legal-trust-notes.md` - TfL, App Store, privacy, data, and trust guidance.
- `10-Exports/` - AI concept reference and exported package assets.

## Most Important Production Notes

- TubeBoard must remain clearly independent from TfL.
- Do not use the official TfL or London Underground roundel.
- Use generated concept imagery as reference only unless cleaned into layered production art.
- Final App Store screenshots must use implemented app states.
- Regenerate final website hero and Open Graph images after real app screenshots are captured.
